# Web-Scraper-Bot
Bot To Scrape A Website. Using axios. cheerio. and tgsnake

# Install

```
git clone https://github.com/Nekozu/Web-Scraper-Bot

cd web-scraper-bot

npm i
```
Fill Token, Api hash, Api id at .env file

Run The Bot
```
node index.js
```
